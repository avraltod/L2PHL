\section*{Overview}

This manual follows the repository layout, moving through the survey lifecycle in
the order the folders present it. The CAPI baseline is frozen, while the CATI
panel rolls forward with each new round. The QC dashboard and the weighting
procedures receive the deepest treatment because they are the live,
hand-over-critical workstreams that a new team must be able to run without
supervision.
